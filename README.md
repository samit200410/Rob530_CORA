# Rob530_CORA
Last Edited Apr 17, 2026.

UMich ROB530 Project by Samit Mohapatra, Kevin Chang, Nicolas Betancur, and Holden Halucha

This repository includes a PlatformIO project for which you need to install the PlatformIO extension in VSCode.

The .ini file configures how the hardware is flashed. Make sure to use the correct USB ports for all of the ESP32s.

Additionally, the MarineRoboticsGroup/cora repository by Alan Papalia is also in the repo so that all the code can be run together at the same time.

Finally, there is code to simulate RSSI measurements and localize from it.
